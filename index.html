Solutions offered by this project


Number of free spots in a certain area

Easy access to the number of free and busy spots in a certain area, providing an easy monitoring of the desired area.

Customizable alerts for occupancy percentages

Alerts via e-mail whenever a certain treshold is surpassed, to indicate the level of occupancy in an certain area.

Availability of a certain parking spot

Easy monitoring of your own parking spot, at any time, from anywhere.

Technologies used in the implementation of this project


JavaScript

High-level, often just-in-time compiled programming language. Used in this project to write the scripts in the Node-Red environment.

Node-Red

Node red is a flow based developement tool for visual programming used for writing together devices, APIs and online services as part of Internet of Things.


AWS IoT

Subservice of Amazon Web Services, an useful tool for computing data into the cloud and work with it.

Amazon Web Services

Subsidiary of Amazon providing on-demand cloud computing platforms and APIs for individuals and companies, on a metered pay-as-you-go basis. Used in this project to process the data, and perform the analysis of it, and graphical representations.


MQTT

MQTT ( MQ Telemetry Transport ) is a lightweight, publish-subscribe network protocol that transports messages between devices. Used to transport data from Node-Red to AWS.

Mail

Mail alerts set by the user in order to know exactly when some monitorized value surpasses a certain treshold, or simply when a parking spot is free.
